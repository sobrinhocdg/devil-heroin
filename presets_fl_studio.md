# Presets Recomendados para FL Studio - Dungeon Synth

Este arquivo contém recomendações de sintetizadores e configurações para usar com os arquivos MIDI gerados pelo AfterRelease33.

## Sintetizadores Recomendados

### 1. Pads e Atmosferas (Drone Pad, Choir Pad, Ambient Drone)
- **Sytrus**: Use presets de pads largos e sombrios.
  - Filtro: Low-pass com ressonância alta.
  - Envelope: Ataque lento, sustain alto.
  - Efeitos: Reverb grande (hall), delay longo.

- **Harmor**: Para texturas harmônicas complexas.
  - Modo: Additive ou FM.
  - Filtro: Band-pass para som etéreo.

### 2. Basses (Pulse Bass, Sub Bass)
- **Sytrus**: Bass synth com sub-bass.
  - Oscilador: Sawtooth ou square wave.
  - Filtro: Low-pass com cutoff baixo.
  - Distortion: Leve para punch.

- **BassDrum**: Para sub-bass pesado.
  - Tune: Baixo para frequências graves.
  - Decay: Longo para sustain.

### 3. Leads e Melodias (Lead Melody, Dark Lead)
- **Sytrus**: Leads com filtro modulante.
  - Oscilador: Sawtooth com PWM.
  - LFO: Modulando filtro para movimento.
  - Efeitos: Chorus, phaser.

- **Harmor**: Para leads etéreos.
  - Modo: FM com operadores complexos.
  - Filtro: High-pass para brilho.

### 4. Arpeggios e Texturas (Arpeggio, Harmonic Texture)
- **Sytrus**: Arpeggiadores com sequências.
  - Arpeggiator: Padrões ascendentes/descendentes.
  - Filtro: Automação para variação.

- **Effector**: Para texturas FX.
  - Modo: Granular ou spectral.

### 5. Bells e Motifs (Bell Motif, Pad Sweep)
- **Bells**: Presets de sino ou chime.
  - Decay: Longo para ressonância.
  - Reverb: Grande para espaço.

- **Sytrus**: Para sweeps harmônicos.
  - Oscilador: Sine waves em harmonia.
  - Filtro: Sweep automático.

### 6. Counterpoint e Sweep
- **Piano Roll**: Para linhas contrapontais.
  - Use piano roll para desenhar linhas melódicas.
  - Efeitos: Reverb e delay.

## Configurações Gerais

### Mixer
- **Volume**: Ajuste para que pads fiquem baixos, bass e lead mais altos.
- **Panning**: Pads no centro, leads e basses com leve pan.
- **EQ**: Corte agudos em pads, boost em mids para bass.

### Efeitos Master
- **Reverb**: Hall grande (2-3 segundos).
- **Delay**: Ping-pong ou tape echo.
- **Compressor**: Para controlar dinâmica.
- **Limiter**: Para evitar clipping.

### Playlist
- **Arrangement**: Use clips longos para sustain.
- **Automation**: Automação de volume, filtro e efeitos para build-up.

## Dicas para Dungeon Synth
- Foque em texturas atmosféricas e graves pesados.
- Use modulação lenta para movimento.
- Adicione ruído branco ou FX para ambiência.
- Mantenha o mix escuro e denso.
